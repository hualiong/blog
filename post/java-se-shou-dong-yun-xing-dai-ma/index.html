【Java SE】一、手动运行代码

2023-01-15 1 min read # Java

学了这么久的 Java，你知道如何手动运行代码吗？🤔

  1. 新建一个目录 src，存放源代码，再写一个简单的HelloWorld类，如下：

    public static void main(String[] args) {
	System.out.println("HelloWorld");
}

Markdown 基本语法

2023-01-08 2 min read # Other

老早就接触 Markdown 语法，听说好多网站都支持就来试试，相关内容可能早就有人发布了，所以该篇文章仅作为学习记录，供自己参考。

注意！该篇文章仅含本人所学的基本语法，内容并不完整。

标题

# + 空格 + 标题，#的个数代表标题的级数。

# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题

字体

斜体 ：* + 文字 + * 或 _ + 文字 + _

粗体 ：** + 文字 + **

删除线 ：~~ + 文字 + ~~

链接 ：[文字] + (地址)

引用

> + 引用内容

分割线

---

图片

![图片名称] + (本地路径 / 图片链接) ：

列表

有序列表

  1. 数字 + . + 空格
  2. 回车后自动生成下一序号
    3. 可以用Tab键进行缩进

无序列表

  • - + 空格
  • 回车后自动生成下一序列
    • 可以用Tab键进行缩进

表格

第一列 | 第二列
-- | --
第一个 | NO.1
第二个 | NO.2
第三个 | NO.3
第一列 第二列
第一个 NO.1
第二个 NO.2
第三个 NO.3

各列之间用 | 隔开，确保每一行的列数是一样的，且每行代码间不能有空行。

代码

单个代码用 ` 符号括起来

代码块则连用```三个符号括起来，同时在首个```后添加语言可以实现语法高亮
